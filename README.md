# Fabrixa API

Official developer resources for the [Fabrixa](https://www.fabrixa.com) Integration API — Europe's reactive textile print platform.

Fabrixa lets merchants, brands, print-on-demand platforms and agencies connect a storefront (Shopify, WooCommerce, custom API) or design in Fabrixa Studio, with reactive-printed apparel, home textiles and fabric-by-the-metre produced and shipped white-label from Portugal, Spain and the Netherlands — MOQ 1.

## In this repo

- **`fabrixa-integration-v2.postman_collection.json`** — a ready-to-import Postman collection covering authentication, orders, fulfilment, Fabrixa Studio, and webhooks against the Fabrixa Integration API (v2).

## Getting started

1. Create an account at https://account.fabrixa.com/signup
2. Create a sales channel in your dashboard to get your Application Access Token and Application Key
3. Import the Postman collection in this repo (`File → Import` in Postman)
4. Set your keys in the collection's Variables tab
5. Start with the `Ping — verify credentials` request under **Auth**
